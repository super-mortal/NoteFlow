## [0.0.1] - 2026-06-12

### Added

- **更新日志归档规范**：新增 `docs/changelog/` 目录，更新日志按语义化版本主版本号（X）分组归档，同 X 系列版本写入同一 markdown 文档（如 `V0.x.x.md`），格式遵循 Keep a Changelog。
- **AI 编写指南**：`docs/changelog/CLAUDE.md` 记录完整的更新日志编写规范，含文件结构、命名规则、格式模板、变更类型和编写要点。
- **根目录指引**：`CLAUDE.md` 新增 Changelog 章节，指引 AI 前往 `docs/changelog/CLAUDE.md` 了解详细规范。

### Changed

#### UI 重构（Phase 1 — 基础框架）

- **顶部导航栏（TopNavbar）**：新增全局导航栏（毛玻璃效果），包含品牌标识、后端健康指示灯（桌面端 Tauri 显示）、设置入口
- **布局重构：三栏 → 两栏**：首页从三栏可拖拽布局（23%+16%+61%）重构为两栏固定布局（300px 侧边栏 + 弹性主内容区），侧边栏可折叠/展开带动画过渡（300ms ease）
- **侧边栏标签切换（SidePanel）**：侧边栏新增「新建笔记」/「生成历史」药丸式标签切换
- **RootLayout**：集成 TopNavbar，整体布局改为 `flex-col h-screen`
- **App.tsx**：移除旧版浮动 `BackendHealthIndicator`（功能已集成到 TopNavbar）
- **主题系统升级**：`index.css` 新增 20+ CSS 变量（`--surface-*` 表面层级、`--gradient-*` 品牌渐变、`--success/warning/error` 语义色、`--text-*` 文字层级）；暗色模式全面优化（降饱和度、增加表面层级区分、半透明毛玻璃）
- **删除冗余文件**：`indexa.css`（与 `index.css` 完全重复）

#### UI 重构（Phase 2 — 核心页面）

- **NoteForm 重构**：
  - 提交按钮改为渐变 `bg-gradient-to-r from-primary to-indigo-500`，增加 Sparkles 图标
  - 平台选择从下拉 Select 改为 `PlatformIconGroup` 图标按钮组（B站/YouTube/抖音/快手/本地）
  - 高级选项（视频理解、笔记格式、备注）折叠在 `CollapsibleSection` 中
  - 模型选择 + 笔记风格保持一行双列布局
  - **所有表单字段、校验逻辑、提交流程保持不变**
- **MarkdownViewer 重构**：
  - 集成 `ViewTabs`（Markdown / 时间轴 / AI 摘要 / 导图）
  - 加载态从 StepBar 升级为 `PipelineProgress` 流水线可视化组件（5 阶段 + 进度条）
  - 空闲态展示 5 步引导图
  - 操作栏使用 `StatusBadge` 展示模型名和风格标签
  - 所有 UI 组件引用 `MarkdownHeader` 移除，内联集成
- **History & NoteHistory 重构**：
  - `History` 简化头部（含记录数统计）
  - `NoteHistory` 搜索框加 Search 图标；使用 `StatusBadge` 替换内联状态文字（SUCCESS→绿色、FAILED→红色、进行中→蓝色脉冲、排队中→黄色）
  - hover 显示删除按钮，添加相对时间显示（xx分钟前/xx小时前）
  - 移除未使用的 `PinyinMatch` 依赖
- **新建组件**：`PipelineProgress`、`ViewTabs`、`PlatformIconGroup`、`CollapsibleSection`、`StatusBadge`

#### UI 重构（Phase 3 — 设置页面）

- **SettingLayout + Menu**：
  - 左侧导航 260px，品牌标识 + 返回首页按钮
  - 菜单项活跃状态显示左侧蓝色指示条 + 背景高亮
  - 移除两个 `index.module.css` 文件，全部迁移到 Tailwind `cn()`
- **Model 页**：
  - 提供商从行列表改为 `grid grid-cols-2` 网格卡片布局
  - `ProviderCard` 从水平行变为垂直卡片：图标在上、开关在右上、名称居中、状态文本在底
  - 搜索框 + 添加按钮联动
  - **数据字段、开关逻辑、路由导航不变**
- **ProviderForm 视觉刷新**：
  - 两栏卡片分区（基本信息 + 模型管理）
  - 使用 Lucide 图标（Save/TestTube/RefreshCw/X 等）
  - 醒目的返回按钮，清晰的字阶和间距
  - **所有表单字段、校验规则、提交逻辑、嵌套路由不变**
- **Transcriber 两栏布局**：
  - `grid grid-cols-1 lg:grid-cols-2`：左栏引擎配置 + 右栏模型管理
  - 自定义模型区集成在右栏
  - **所有配置项、保存逻辑、下载/添加/删除功能不变**
- **Monitor 仪表盘风格**：
  - 2×2 网格卡片展示四个组件状态（后端/GPU/Whisper/FFmpeg）
  - 本地状态标签替换 `Badge` 组件
  - **刷新逻辑、API 调用、自动轮询 30s 不变**

#### UI 重构（Phase 4 — 交互打磨）

- **主题切换**：`ThemeToggle` 组件集成到 TopNavbar，`localStorage` 持久化，监听 `prefers-color-scheme`
- **CSS 动画**：新增 `animate-fade-in-up` 和 `animate-scale-in` 两个自定义 keyframe 动画，主页内容区渐入
- **骨架屏加载态**：新建 `Skeleton` 组件，Transcriber / Monitor / ProviderForm 加载态改为骨架屏布局
- **ChatPanel UI 统一**：颜色方案改为 `var(--primary)` 品牌色，头像圆角优化，边框样式统一
