## [0.2.0] - 2026-06-14

### Changed

- **统一数据目录到 `data/` 下**：所有用户数据（笔记、下载缓存、截图、模型、导出、向量数据库、SQLite）收敛到 `<安装目录>/data/` 统一管理，不再散落在根目录或系统 `%APPDATA%`

  - `data/notes/` — 原 `note_results/`（Markdown 笔记、JSON 缓存、status 文件）
  - `data/downloads/` — 原 `data/`（yt-dlp 下载缓存）
  - `data/screenshots/` — 原 `static/screenshots/`（视频截图、封面）
  - `data/models/` — 原 `%APPDATA%/NoteFlow/models/`（Whisper 模型，迁入统一目录）
  - `data/exports/` — PDF/DOCX 导出
  - `data/vector_db/` — 原 `vector_db/`（ChromaDB 向量索引）
  - `data/noteflow.db` — 原根目录 SQLite 数据库
- **同步更新所有引用路径**：`main.py`、`engine.py`、`sqlite_client.py`、`note.py`、`routers/note.py`、`chat_tools.py`、`vector_store.py`、`universal_gpt.py`、`video_helper.py`、`export.py` 共 10 个后端文件的默认路径统一改为 `data/` 下
- **更新配置文件**：`.env.example`（根目录 + backend）新增 `DATABASE_URL`、`VECTOR_DB_DIR`，所有路径变量默认值改为 `data/` 下；`Dockerfile.complete` supervisor 环境变量同步；`docker-compose.yml` 注释同步；`.gitignore` 合并为统一的 `/backend/data/` 规则
- **更新部署文档**：`Docker本地部署说明.md` 数据目录树同步为新的 `data/` 结构

### Added

- **桌面端安装支持自定义路径**（`noteflow_frontend/src-tauri/tauri.conf.json`）：
  - 新增 `bundle.windows.nsis` 配置：`installMode: "currentUser"`（安装到 `%LOCALAPPDATA%\NoteFlow`，无需管理员权限），`displayLanguageSelector: true`（安装器显示语言选择页）

## [0.0.10] - 2026-06-13

### Fixed

- **修复原片截图功能不生效**：`backend/app/routers/note.py`
  - 根因：前端通过 `format: ['screenshot']` 表达截图意图，但后端判断是否在 Prompt 中插入 `*Screenshot-[mm:ss]` 标记时依赖的是独立布尔字段 `screenshot`，该字段在前端未绑定 UI 始终为 `false`，导致 GPT 无法在笔记中添加截图标记，后端也无法替换为真实截图
  - 修复：在 `run_note_task` 中添加自动推断逻辑 — 若 `format` 数组包含 `'screenshot'` 则自动将 `screenshot` 设为 `true`，`link` 同理
  - 效果：勾选"原片截图"后，AI 会在笔记中插入 `*Screenshot-[mm:ss]` 标记，后端用 FFmpeg 截取视频帧替换为真实图片展示在笔记中

  - 右侧操作区：设置图标 + 返回首页按钮

### Fixed

- **修复平台图标尺寸错乱**：SVG 图标组件（`BiliBiliLogo`、`YoutubeLogo`、`DouyinLogo`、`KuaishouLogo`、`LocalLogo`）存在严重显示问题
  - **问题根源**：SVG 内部硬编码 `width="200" height="200"` 与外部容器 CSS `h-4 w-4`（16px）冲突，导致彩色块重叠、图标巨大化

    - 移除所有 SVG 的 `width`、`height`、`version`、`t`、`p-id` 等冗余属性

    - 保留 `viewBox="0 0 1024 1024"` 用于自适应缩放
    - 添加 `className="h-full w-full"` 使 SVG 正确继承容器尺寸
  - **影响文件**：`BillNote_frontend/src/components/Icons/platform.tsx`
- **优化图标容器布局**：`PlatformIconGroup` 的图标容器添加 `flex items-center justify-center`，确保 SVG 在 `h-4 w-4` 容器内居中显示
