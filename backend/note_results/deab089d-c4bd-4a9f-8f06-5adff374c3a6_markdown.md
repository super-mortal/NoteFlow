## 智慧教育平台 - 项目演示笔记

### 1. 项目概述

- **参赛信息**：传智杯国赛，AR月开发挑战赛赛道
- **作品名称**：智慧教育平台
- **作者**：梁天伟，广州华南商贸职业学院
- **目标用户**：小学到高中学生
- **核心理念**：覆盖诊断 → 规划 → 练习 → 反馈 → 复盘的学习闭环
- **部署状态**：因NAT服务器证书问题及服务商维护通知，本次国赛采用本地环境演示

### 2. 技术架构

- **前端**：Vue 3 + Element Plus（单页应用），Vue Router（路由管理），Axios（请求接口）
- **后端**：Node.js（运行环境）
- **数据库**：MongoDB（本地连接）
- **用户认证**：JWT（JSON Web Token）
- **AI能力**：接入OpenAI兼容的大模型，实现：
  - 流式对话
  - 知识库RAG检索
  - AI学习计划生成
- **实时通知**：Socket.IO（实时推送）

### 3. 主要功能模块

#### 3.1 首页

- 核心功能区域、**"为什么选择我们"** 区域、常见问题区域
- 底部包含：隐私政策、服务条款、使用文档教程、产品介绍

#### 3.2 用户注册与登录

- 仅支持QQ邮箱注册（安全考虑）
- 注册成功后会触发实时通知（系统默认开启邮箱通知）

#### 3.3 助手与知识库（AI对话）

- **布局优化**：采用左右布局，新增助手功能（非单一AI对话框）
- **上下文管理**：每次消息携带前10条历史记录，支持多轮对话记忆
- **知识库**：
  - 支持文件和链接添加（使用向量化嵌入模型处理）
  - 支持文本添加（Markdown编辑器，左编辑右预览）
  - 对话时可临时关联知识库，优先检索知识库内容
- **输出格式**：支持Markdown渲染，重要内容加粗

#### 3.4 错题管理

- 新增日历视图，每日统计错题数量
- 支持按科目筛选、掌握/未掌握状态筛选
- **AI分析**：调用接口进行实时流式分析错题数据
- 学习科目在个人中心编辑，添加错题时选择对应科目

#### 3.5 学习计划

- **全新自动化定制**：AI在后台收集用户画像（年级、学习科目、薄弱/强势科等）
- 支持设置定时自动生成（每日指定时间），生成后通过邮箱和实时通知推送
- 计划以**时间线**形式展示，点击可展开查看详情，支持编辑和删除
- 支持手动立即执行AI生成，可指定科目、计划周期等

#### 3.6 笔记功能

- 基于Markdown编辑器，左编辑右实时预览
- 支持文件夹分类与嵌套，右键重命名
- 支持笔记内容搜索、排序方式设置

#### 3.7 学习社区

- **重大重构**：支持问题交流、社交功能、AI分析
- 左侧筛选：全部问题、最新问题（平台下最新10条）、关注的人的问题、我的问题、我的收藏
- 问题详情页可嵌套回答，方便用户交流
- 个人中心信息可通过社区左下角快捷跳转
- AI可分析社区热门话题及个人学习问题

#### 3.8 反馈功能

- 用户提交反馈后，管理员后台实时收到通知
- 管理员回复后，用户可查看回复

#### 3.9 成绩管理

- 支持添加考试成绩，并内置演示数据
- 添加成功后进行AI实时分析（支持Markdown及流式输出），分析结果可保存至左侧方便查看
- 包含成绩图表分析

#### 3.10 后台管理

- 新增系统公告功能：管理员可立即发送公告，指定用户（演示中发送给5位用户），用户端实时收到通知
