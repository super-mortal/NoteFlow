## 美国原生家宽住宅IP免费领取教程

### 1. 资源概述
- **资源类型**：美国原生家宽（Native Home Broadband）住宅IP。
- **资源特点**：
  - **极度纯净**：通过双ISP（Internet Service Provider，互联网服务提供商）检测，ASN显示为双ISP类型。
  - **原生家庭宽带**：非数据中心IP，具有极高的隐蔽性和真实性。
  - **流量限制**：无无限流量限制说明，但为短效IP。
  - **数量**：每个CDK兑换后可提取 **3个** IP资源。
  - **有效期**：每个IP提取后使用时长为 **1到2小时** 不等。

### 2. 账号注册与CDK兑换
1. **访问官网**：进入相关服务提供商官网。
2. **注册账号**：完成用户注册并登录。
3. **进入仪表盘**：登录后，从页面右上角进入用户仪表盘（Dashboard）。
4. **兑换CDK**：
   - 在仪表盘右侧找到 **CDK兑换** 区域。
   - 点击输入框，粘贴或输入获取到的CDK代码。
   - 点击兑换按钮完成激活。

### 3. IP提取步骤
1. **进入IP管理页面**：
   - 在左侧导航栏中，点击 **“短效住宅IP账密模式”**。
   - 页面将显示当前可用的IP余额（即兑换获得的3个名额）。
2. **筛选条件**（可选）：
   - **国家/地区**：选择需要提取的国家（如美国）。
   - **ASN筛选**：可通过ASN（Autonomous System Number，自治系统号）进行更精细的筛选，以确保获取特定ISP的资源。
3. **执行提取**：
   - 从下方列表中选择 **一个或多个** IP资源。
   - 点击提取按钮。
4. **获取凭证**：
   - 提取成功后，IP信息会显示在 **“使用列表”** 中。
   - 点击右侧的 **复制按钮**，复制IP的使用信息（通常包含IP地址、端口、用户名和密码）。

### 4. 配置与使用演示（以指纹浏览器为例）
1. **新建业务端口**：
   - 在指纹浏览器中创建一个新的业务配置文件或端口。
2. **设置代理信息**：
   - 下滑找到 **代理信息（Proxy Settings）** 设置区域。
   - **代理类型**：选择 **S5**（SOCKS5）。
   - **粘贴凭证**：将刚才复制的使用信息粘贴到对应的IP、端口、用户名和密码栏中。
3. **启动端口**：
   - 完成设置后，打开并启动该业务端口。

### 5. 纯净度检测验证
为确保资源质量，建议使用以下网站进行检测：

- **检测网站 IPIP**：
  - **结果**：ASN显示为 **双ISP**，类型为 **原生家庭宽带**。
  - **结论**：纯净度非常干净。

- **检测网站 APO**：
  - **结果**：同样检测为 **原生高质量住宅资源**。
  - **结论**：极度纯净，符合预期。

### 6. 注意事项
- **时效性**：提取后的IP有效期较短（1-2小时），建议按需提取，避免资源浪费。
- **网络环境**：确保本地网络环境稳定，以便顺利连接和使用代理。
- **多工具验证**：若对单一检测结果不放心，建议使用多个检测网站（如IPIP、APO等）交叉验证IP的纯净度。
