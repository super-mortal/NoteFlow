# 电影《保你平安》笔记

## 影片概要
- 上映时间：2022年
- 类型：温情喜剧片
- 核心主题：造谣成本与澄清代价、网络暴力、人间温情

## 主要人物
- **魏平安（大鹏饰）**：主角，外号“吉安段尾虎”，曾因替兄弟出头入狱5年，后在陵园做销售。
- **韩露**：已故客户，被谣传为“坐台小姐”，实际是脑瘤去世的爱心人士，临终前裸捐136万给福利院。
- **魏如意**：魏平安的妹妹，在密室逃脱做NPC，协助查谣。
- **冯总**：C05墓主家属，因谣言要求迁走韩露的墓，威胁取消与陵园的合作。
- **托尼**：宠物店老板，谣言传播者之一。
- **老金**：吉昭国际董事长，从贴吧看到谣言后传播。
- **完美先生421**：网名，在贴吧编造韩露的谣言，只为获得积分免费看小说。
- **魏魔力**：魏平安的女儿，因支持父亲而遭遇校园霸凌。

## 核心情节（精简版）

### 起因：谣言引发迁坟风波
- 魏平安卖出的C06号墓地客户韩露，被隔壁C05墓主冯总投诉是“坐台小姐”，要求陵园迁坟否则取消合作。
- 魏平安坚信韩露清白，决定追查谣言源头。

### 追查过程：层层溯源
- **宠物店**：托尼称信息来自门口发传单的人偶。
- **密室逃脱**：通过妹妹协助，找到人偶杨玉婷，其上线是母亲、舞伴、最终指向吉昭国际的老金。
- **吉昭国际**：老金称在贴吧看到“完美先生421”的留言，称韩露“捐的钱都是睡出来的”。
- **内蒙古赤峰**：魏平安找到“完美先生421”，对方承认完全不认识韩露，只为评论热帖获得积分（三个月亮船可免费读一章小说），杜撰了谣言。

### 高潮：冒雨抢骨灰并寻找真相
- 冯总不顾下雨强行要迁坟，魏平安抢走弟弟的骨灰盒，驾车前往赤峰。
- 途中女儿因举报校园霸凌被报复，魏平安一度动摇，但女儿鼓励他坚持。
- 最终在车站抓到“完美先生421”，真相大白。

### 结局：真相与回馈
- 谣言澄清，冯总撤回迁坟要求。
- 魏平安因偷骨灰等行为罚款、赔偿，损失了自己的虎奔车，改骑电动车。
- 生日当天，韩露生前委托送他一份礼物——在江边绽放的“666大礼花”（烟花）。

## 经典台词与细节
- “图个人间自有真情在。当你张嘴说一个女人是小姐的时候，不管她是不是，她都已经了。”
- 女儿魏魔力：“我的事儿已经做完了，你也要加油。”
- 韩露小时候在福利院，生日想喝可乐，遇到小魏平安（成年后）给她买了可乐。

## 影片点评
- 笑点与泪点并存，荒诞故事直击社会痛点：造谣成本极低，澄清代价极大。
- 呼吁理性看待网络信息，抵制无端造谣。

## 关键数据
- 韩露裸捐金额：136万。
- 造谣动机：贴吧用户为赚积分（3个月亮船=免费读一章小说），编造“睡过韩露”的留言。
